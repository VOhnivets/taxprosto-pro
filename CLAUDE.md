# taxprosto.pro

## Продукт
Вебзастосунок для самостійного заповнення та подачі річної податкової декларації (форма 0100424) громадянами України.

**Ключовий flow:**
1. Користувач завантажує довідку про доходи з Дії (PDF/XML)
2. Застосунок парсить дані (роботодавець, нарахований дохід, утриманий ПДФО/ВЗ)
3. Формує XML-файл за схемою ДПСУ
4. Користувач завантажує XML → імпортує в Електронний кабінет (cabinet.tax.gov.ua)

## Аудиторія
Фізичні особи — платники податків в Україні, що подають річну декларацію самостійно: ФОП 3 група, фізособи з іноземними доходами, ті що мають право на податкову знижку.

## Стадія
MVP — перший робочий прототип. Пріоритет: парсинг довідки → генерація валідного XML → завантаження. Декларація за 2024 рік, форма 0100424.

**Домен**: https://taxprosto.pro/

---

## Технічний контекст

- **Схема XML**: форма J0100424 / F0100424 (ДПСУ, актуальна версія 2024–2025)
- **Джерело даних**: довідка 3-ПДФО або виписка з Дії (PDF або структурований формат)
- **Цільова система**: cabinet.tax.gov.ua → "Звітність" → "Імпорт XML"
- **Валідація**: XSD-схема ДПСУ (публічна, є на tax.gov.ua)
- **Архітектура**: 100% client-side на старті — жодних серверів, жодного збереження даних

## Структура папок

- `knowledge/` — XSD-схеми, зразки XML, документація ДПСУ, приклади довідок
- `output/` — прототипи, згенеровані XML для тестів, скріншоти UI
- `.claude/plans/` — плани реалізації фіч
- `.claude/skills/tax-consultant/` — skill податкового консультанта для підтримки розробки
