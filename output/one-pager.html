taxprosto.pro

Заповни річну декларацію самостійно — без бухгалтера, без помилок.

Проблема

Мільйони українців щороку зобов'язані подавати річну податкову декларацію (форма 0100424), але більшість не знають як — або платять бухгалтеру за просту операцію. Офіційний Електронний кабінет незручний і вимагає технічних знань.

Рішення

Веб-застосунок, який перетворює довідку про доходи з Дії у готовий XML-файл для подачі. Все відбувається в браузері — жодних серверів, жодного витоку даних.

  1. Завантажуєш довідку про доходи (PDF або XML з Дії)
  2. Застосунок парсить дані: роботодавець, нарахований дохід, утриманий ПДФО і ВЗ
  3. Генерує валідний XML за схемою ДПСУ (J0100424 / F0100424)
  4. Завантажуєш файл і імпортуєш у cabinet.tax.gov.ua — готово

Для кого

ФОП 3-я група Подають річну декларацію з доходів поза ЄП
Фізособи з іноземними доходами Зарплата, дивіденди, фріланс з-за кордону
Податкова знижка Навчання, іпотека, лікування — хочуть повернути переплату
Самозайняті та інші Будь-хто, хто подає форму 0100424 самостійно

Ключові переваги

  • Приватно: дані не залишають браузер — 100% client-side
  • Валідно: XML перевіряється по XSD-схемі ДПСУ
  • Просто: без реєстрації, без оплати на старті, без зайвих кроків
  • Сумісно: результат напряму імпортується в Електронний кабінет

Технічний стек

Client-side JS XML / XSD ДПСУ PDF / XML парсинг Дія API cabinet.tax.gov.ua

Стадія

MVP — перший робочий прототип. Пріоритет: парсинг довідки → генерація валідного XML → завантаження. Декларація за 2024 рік, форма 0100424.

Наступні кроки

  • Реалізувати парсер PDF-довідок з Дії
  • Генерація та валідація XML по XSD-схемі
  • Публічний запуск з feedback-формою
  • Розширення: податкова знижка, іноземні доходи, нагадування
